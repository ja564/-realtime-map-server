# 实时事件地图项目

这是一个使用 MapLibre GL 库构建的实时事件地图项目。用户可以在地图上标记事件，并查看其他用户提交的事件。

## 项目结构

```
realtime-map-project
├── client
│   ├── index.html      # 主HTML文件，包含页面结构和样式链接
│   ├── style.css       # 项目的样式定义
│   ├── app.js          # JavaScript逻辑文件，处理地图交互和事件标记
│   └── README.md       # 项目的文档，提供使用和运行说明
└── README.md           # 项目的根文档，包含项目概述和安装说明
```

## 安装和运行

1. 克隆项目到本地：
   ```
   git clone <项目的Git仓库地址>
   ```

2. 进入项目目录：
   ```
   cd realtime-map-project
   ```

3. 打开 `client/index.html` 文件，使用浏览器查看项目。

## 使用说明

- 在地图上点击以标记新事件。
- 在弹出的模态框中输入事件描述，然后提交。
- 事件将被标记在地图上，供其他用户查看。
